Documentation!

This website can be accessed in three different ways. One who have already registered can directly enter into this site. But one who have not can register and one can also go to book store directly.

If you are an already registered user, you can follow the following steps:
  • You can find that so many boxes that have appeared, with ISBN,Title,Goodreads rating, Author and the year of publication in it.
  • You will also find a search box where you can search for books by entering their ISBN or Author or Year or Title. (Note: The queries are case sensitive.Its preferrable to search for a part of the what you know and you'll get better results by searching with small lettered queries.)
  • When you click on any of that boxes, you'll be redirected to a page showing the details of a book.
  • Then, you'll find that there will be a drop-down list showing numbers from 1-5 and a rate button nearby. If you select any of that values and then rate a book, then your rating will be successfully added.
  • You can also see a 'View Comments' button. When you click on that, you'll then see a page showing all the comments on that book. There's also a textarea space provided for you, inorder to post any comment.
  • If none of them commented on a book, then it'll show a message displaying, 'No one has commented on this book yet!'
  • If you want to logout, then you'll see an icon showing the logout symbol. Logout by clicking on that.
  • If you signed in into the site and ten forgot to logout, then the next time you go to the login URL you'll find that you are automatically redirected to book store!
  • If you want to go to book store, click on the icon depicting books.
  • Note: When you load the book store, you'll find that it takes time to load. This is because, it takes the latest updated ratings on the books from goodreads.
If you are not a registered user
  • It is preferred to be registered user, because else you won't get all features of this site.
  • You'll see all the features mentioned above but one thing is that you are neither able to rate a book nor comment on a book.





Developer Tools

This site has API's which are in the format, 'book/api/(isbn number)'. You will be getting a JSON object (JavaScript Object Notation) containing the details of a book. For example: { "title": "Memory", "author": "Doug Lloyd", "year": 2015, "isbn": "1632168146", "review_count": 28, "average_score": 5.0 }

If you are trying to invoke an invalid isbn number, then automatically a 422 error will be displayed.
